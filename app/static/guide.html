📊 Excel 统一分析工具

PPT 报告生成 + 透视分析 · 完整操作指南

一、快速入门

三步完成分析
  1. 准备 Excel 配置文件（项目配置.xlsx），内含 PPT配置 和 透视分析 两个 Sheet
  2. 准备好数据文件，与配置文件放在同一文件夹内
  3. 打开工具 → 选择文件夹 → 点击 「开始分析」，自动生成 PPT 报告 + 透视分析 Excel

二、功能概览

一个 项目配置.xlsx 可同时包含两类配置：

PPT配置 Sheet 18页 · 25个图表

定义 PowerPoint 幻灯片布局、图表类型、数据来源等。支持 10 种图表类型（column / line / pie / doughnut / area / scatter / combo / map / heatmap / 多图表混合）。

透视分析 Sheet 23个分析任务

定义数据透视/分组统计/占比分析/多表关联等任务。支持交叉表、分箱、值计算、LEFT/RIGHT JOIN 等高级功能。

数据源—数据文件名称，支持 Excel(.xlsx) 或 CSV 文件，系统按匹配度自动选择。填 {pivot} 或 透视结果 可引用透视分析的输出结果
X轴范围✔列名（频段）、多列（频段,带宽）或 Excel 范围（A2:A15）
Y轴范围✔列名或逗号分隔多系列，也支持 Excel 范围
图表标题—图表上方显示标题
颜色—单系列颜色（如 #C8102E），不填则自动调色板
区块名—在数据 Sheet 中定位特定区块的表头行（CSV文件不支持）
结论模板—含占位符的结论文本，见下方「结论模板」
数据源匹配说明：系统支持 Excel(.xlsx) 和 CSV 两种数据文件格式。填写数据源时：
  • 填写文件名（带扩展名）：如 网络指标数据.xlsx 或 销售数据.csv
  • 填写文件名（不带扩展名）：系统自动匹配，如 网络指标数据
  • 填写部分名称：系统按匹配度自动选择，如 指标 可能匹配到 网络指标数据.xlsx
  • 不填写：继承上一行的数据源
💡 引用透视分析结果：如果配置包含「透视分析」Sheet（综合配置），PPT 图表可以直接引用透视分析的输出结果作为数据源：
  • 在「数据源」列填写 {pivot} 或 透视结果 或 pivot
  • 在「数据Sheet」列填写透视分析任务中「结果Sheet」的名称
  • 执行顺序：系统会先执行所有透视分析任务，再生成 PPT，确保结果可用
  • 适用场景：需要先汇总/分组数据，再用汇总结果做 PPT 图表

  • CSV 文件：忽略 Sheet 参数，直接读取 CSV 数据

映射列只需填 新显示名，按顺序依次对应行维度 / 列维度 / 值字段。

行维度行映射 →

4.2 聚合方式

支持的类型
配置值效果
sum求和（默认）

nunique / 去重计数去重计数
pct / 占比所占比例（合计100%）

聚合方式与值字段的对应规则：
• 聚合数 = 值字段数：一对一映射
• 聚合数 < 值字段数：剩余值字段默认 sum
• 聚合数 > 值字段数（如单值字段多聚合）：全部聚合方式应用到该字段

4.3 占比（pct / count_pct）

聚合方式支持两种占比计算：

数值占比 占比 / pct

= sum(值) / sum(所有值)

行维度值聚合
频段上行PRB利用率占比

→ 各频段利用率之和占总体利用率的百分比

计数占比 计数占比 / count_pct

= count(行) / count(所有行)

行维度值聚合
频段上行PRB利用率计数占比

→ 各频段数据行数占总行数的百分比

提示：两种占比也可以与 过滤条件 配合使用，先筛选再计算占比。例如只统计 n78 频段各地区的数据行数占比。

4.4 分箱（连续值离散化）

在 「分箱」列填写规则，支持两种模式：

自动等距 3 个参数
字段=最小值,最大值,箱数

例：5G用户数=0,30000,4
→ 0~7500, 7500~15000, 15000~22500, 22500~30000

自定义断点 4+ 个参数
字段=断点1,断点2,断点3,...

例：下行PRB=0,30,50,70,100
→ 0~30, 30~50, 50~70, 70~100

规则：3 个参数 = 自动等距；4 个及以上 = 自定义断点。分箱后列自动替换行/列维度中的原始列名。

4.5 值后处理（聚合后换算）

聚合完成后对数值列做一步换算，在 「值计算」列填写：

表达式效果示例
*100乘以 10077.4% → 7740
/1000除以 10005500 → 5.5
+10加 10400 → 410
-5减 5100 → 95

多值字段对应多表达式时逗号分隔：*100,*1000（利用率×100，速率×1000）。

4.6 多表关联（JOIN）

在 「数据源」列填写 JOIN 表达式：

基础 JOIN
文件A.xlsx JOIN 文件B.xlsx ON 键=键

系统自动扫描目录，支持同文件不同 Sheet 自动匹配。

跨文件 + 指定 Sheet
A.xlsx@指标 JOIN B.xlsx@地理 ON ID=ID

用 @Sheet名 显式指定读哪个 Sheet。

写法说明
A.xlsx JOIN B.xlsx ON k=kINNER JOIN（默认）
A.xlsx LEFT JOIN B.xlsx ON k=k保留左表全部行
A.xlsx RIGHT JOIN B.xlsx ON k=k保留右表全部行
A.xlsx@S1 JOIN B.xlsx@S2 ON k=k跨文件 + 各自不同 Sheet

关联后的数据按你配置的行/列/值维度做透视，和普通单表完全一致。

4.7 过滤条件

在 「过滤条件」列填写表达式，在透视聚合之前对原始数据进行筛选。系统优先使用 pandas query() 语法，若失败则使用内置解析器。

支持的运算符
运算符说明示例
=等于频段='n78'
!=不等于频段!='n41'
> / >=大于 / 大于等于上行PRB利用率>60
< / <=小于 / 小于等于下行速率<=100
in ()属于集合地区 in ('北京','上海','广州')
包含字符串包含小区名称包含'城区'
多条件组合
连接词说明示例
AND同时满足频段='n78' AND 上行PRB利用率>60
OR满足其一地区 in ('北京','上海') OR 频段='n41'
()括号分组(频段='n78' AND 上行PRB利用率>60) OR 地区='北京'
完整示例
# 只分析 n78 频段中利用率超过 60% 的数据
频段='n78' AND 上行PRB利用率>60

# 分析北京或上海，或 n41 频段的数据
地区 in ('北京','上海') OR 频段='n41'

# 名称包含"城区"且下行速率低于 100 的记录
小区名称包含'城区' AND 下行速率<100
提示：
  • 字符串值需要用英文单引号或双引号包裹：频段='n78'
  • 中文引号（''）会自动转换为英文引号
  • 数值比较不需要引号：上行PRB利用率>60
  • 过滤在聚合前执行，可配合占比计算实现「先筛选再统计占比」
  • 若过滤后数据为空，任务会报错跳过

4.8 配置校验

执行透视分析前，系统自动校验配置项，提前发现错误。也可使用 --check 参数仅校验不执行。

校验项一览
级别校验内容说明
error值字段为空必须指定至少一个值字段
error聚合方式不支持检查是否填写了合法的聚合函数名
error数据源文件不存在指定的数据文件在目录中找不到
errorSheet 不存在数据文件中没有指定的 Sheet
error列名不存在行维度 / 列维度 / 值字段中的列在数据中找不到
warning行维度和列维度都为空将生成单行汇总，可能非预期
warning值映射数量不一致值映射数与值字段数不匹配
info已设置为不计算「是否计算」列为否，该任务将被跳过
使用方式
# 仅校验配置（不执行分析）
python app/main.py pivot 路径/文件夹 --check

# 正常执行时自动校验（有 error 则中止）
python app/main.py pivot 路径/文件夹
校验规则：
  • error（错误）：必须修正才能继续执行
  • warning（警告）：可继续执行，但可能影响结果
  • info（提示）：仅供了解，不影响执行
  • 设置「是否计算」为否的任务仅产生 info 提示，不进行其他校验

五、命令行用法

# 无参数 → 打开 GUI 窗口
python app/main.py

# 自动检测配置并执行（PPT + 透视分析）
python app/main.py 路径/文件夹
python app/main.py 路径/项目配置.xlsx

# 仅执行 PPT 生成
python app/main.py ppt 路径/文件夹

# 仅执行透视分析
python app/main.py pivot 路径/文件夹

# 仅校验透视分析配置（不执行）
python app/main.py pivot 路径/文件夹 --check

输出文件保存在 output_时间戳/ 子目录。PPT 为 .pptx，双击图表可查看和编辑原生数据。

❓ 常见问题（FAQ）

命令行路径包含空格会报错吗？

GUI 模式：完全没问题，通过对话框选择的路径会自动处理空格。
命令行模式：需要给路径加双引号，例如：

python app/main.py "F:\我的 文件夹"      # ✅ 正确
python app/main.py F:\我的 文件夹        # ❌ 错误，空格会拆分参数

文件读取本身（pandas、openpyxl）完全支持空格路径。

数据文件一定要和配置文件放同一个文件夹吗？

不一定。工具会自动在同一文件夹中查找第一个非配置 Excel/CSV 文件作为数据源。也可以在配置文件中通过「数据源」列指定其他文件的名称或路径。

透视分析和 PPT 报告可以同时生成吗？

可以。如果配置文件中同时包含「PPT配置」和「透视分析」两个 Sheet，工具会自动识别并依次生成 PPT 和 Excel 分析结果。

Excel 文件被占用导致读取失败？

请确保配置和数据 Excel 文件没有被其他程序（如 Excel 本身）打开状态。工具会尝试读取文件，文件被占用时可能报错。

CSV 文件不支持哪些功能？

CSV 文件不支持以下功能（Excel 支持）：

  • 「区块名」列（用于在同一 Sheet 中划分数据区域）
  • 多 Sheet 结构的数据文件

其他功能（透视分析、PPT 图表生成）均支持。

图表数据源如何指定不同文件？

在 PPT 配置的「数据源」列填写文件名即可：

  • 带扩展名：销售数据.csv
  • 不带扩展名：销售数据（系统自动匹配）
  • 模糊匹配：填写部分名称如 指标，系统按匹配度自动选择
值映射数量和值字段数量不一致怎么办？

值映射数量应等于输出列数（非原始值字段数）。单值字段多聚合（如 sum,pct）会生成多列，此时值映射数量可能超过值字段数。如果值映射少于输出列数，剩余列保留原名。同名值映射会自动添加聚合后缀区分（如 统计 → 统计_sum）。

单值字段多个聚合方式只有一列输出？

已修复。当值字段只有 1 个、聚合方式有多个时（如值字段=基站数、聚合=sum,mean,pct），所有聚合方式都会应用到该字段，生成对应数量的列。
